Setup guide
1
Add your tasks
On the Log tab, tap ＋ to add a good task or a bad one to avoid. Give it a name, points value, and icon.
2
Set active days
In the task editor, choose which days of the week the task applies to. Unscheduled days won't break your streak.
3
Log each day
Tap + to log a task. Tap − to undo. Good tasks add XP; bad tasks deduct XP when logged.
4
Add reminders
Hold a task card to edit it, then toggle the 🔔 Reminder switch. Choose once daily, weekdays, weekends, or hourly between set times.
5
Install as an app
In your browser menu, tap "Add to home screen" to install Streak as a PWA. The Install button above will appear if your browser supports it.
6
Back up your data
Use Export in the Data section to save a JSON backup. Import it on a new device or after clearing the app.
